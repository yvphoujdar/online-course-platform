🎓 EduStream — Modern Course PlatformEduStream is a high-performance, Single Page Application (SPA) designed to provide a premium, seamless learning experience. Moving away from slow, static templates, this project features a "Glassmorphism" aesthetic with instantaneous navigation and real-time data filtering.
 Live DemoVercel/GitHub Pages: https://online-course-platform-taupe-omega.vercel.app/
GitHub Repository: https://github.com/yvphoujdar/online-course-platform
✨ Key Features
Dynamic Course Grid: An automated layout that renders available tracks based on live data filtering.
Smart Filtering Engine: Instant category sorting for Programming, Marketing, Data Science, and Business without page reloads.
Curriculum Transparency: Interactive accordion-style components that allow users to expand and preview detailed course modules.
Morphing UI Logic: A dynamic header system that transforms the page identity based on the selected category.
SaaS-Style Enrollment: Streamlined, validated registration forms featuring professional input-focused states and "Processing" feedback.
Save for Later: A persistent engagement tool that allows users to bookmark specific courses throughout their session.
 Technical StackFrontend: HTML5, CSS3, Vanilla JavaScript.
Design: Custom "Glassmorphism" UI with frosted glass effects and fluid CSS Grid/Flexbox layouts.
Build & Deployment: Optimized for speed and hosted via Vercel/GitHub Pages for global edge delivery.
Architecture: Modular component-based architecture for a scalable and clean codebase.
 Working Logic
The platform operates as a high-performance SPA leveraging a centralized JavaScript data structure to manage course content dynamically. When a user interacts with the interface—such as selecting a category or searching—the system executes real-time DOM manipulation to update the view without a full page reload. This logic ensures that every interactive element, from the custom-styled dropdowns to the modal pop-ups, scales perfectly across mobile and desktop resolutions.
